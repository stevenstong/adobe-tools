const URI & uri,
bool bUIName 
) const
virtual

Called to get a resource's long name. \ For example, the long name for a file resource is the file's full path.

Parameters
ref[IN] UID reference of the link resource.
uri[IN] URI of the resource to get a long name for.
bUIName[IN] Denotes to return the resource's long UI name if different from the resource's actual long name.\ In some cases, resources may want to have long UI name that differs from the resource's actual long name. For example, file resources on the Mac return a POSIX path as the resource's actual long name, and a HFS path as the resource's long UI name.
Returns
Resource's long name.
