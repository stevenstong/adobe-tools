Detailed Description

This interface defines utility methods for accessing a media player.

const WideString & mediaURL 
)
pure virtual

Tests if a player can obtain a poster from a specified URL. Posters may not be created for a variety of reasons, such as internal player error, security restriction or the media does not have a visual representation (pure audio media does not).

Parameters
mediaTypespecifies the type of media.
mediaURLis a valid URL currently loaded in the player.
Returns
true if a poster can be created; false otherwise.
